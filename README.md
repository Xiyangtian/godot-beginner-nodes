# Godot入门-节点文档项目

这是一个Godot游戏引擎节点系统的入门文档网站。

## 项目介绍
该项目包含Godot引擎中常用节点的详细文档和使用示例，帮助初学者快速掌握Godot节点系统。

## 技术栈
- HTML5
- CSS3
- JavaScript
- Godot Engine

## 在线访问
GitHub Pages地址：https://xiyangtian.github.io/godot-beginner-nodes/

## 本地运行
1. 克隆仓库：`git clone https://github.com/Xiyangtian/godot-beginner-nodes.git`
2. 在浏览器中打开 `index.html` 文件

## 开发说明
- `index.html` - 主页面
- `styles.css` - 样式文件
- `script.js` - JavaScript功能
